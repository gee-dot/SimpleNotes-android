# SimpleNotes Android App

A basic Android notes app built with Java.

## Features
- Accepts user input
- Shows multiple notes
- Simple design
- Input validation

## Tech Stack
- Android Studio
- Java
- Android Views (EditText, Button, TextView)

## How to Run
1. Clone this repository.
2. Open in Android Studio.
3. Build and run on an emulator or device with API 23+.

## What I Learned
- Android UI basics
- Event handling
- Lists and string handling
- Git & GitHub basics

##  Notes
This app currently does not persist notes between launches.
